How to solve Uknown Server Error?

1. Go to search in windows 🔍 and search view network connections 🖥️.

2. Right click your network connection 🛜 and select properties ⚙️.

3. In that scroll down until you see internet protocol version 4 ✅ and version 6✅, now double click internet protocol ver 4 (do not disable ver 4 or your internet will be lost 🔻).

4. Click on the option "Use the following DNS server addresses" and in that type:
Preffered DNS Server: 1.1.1.1
Alternate DNS Server: 1.0.0.1

5. Then click ok.
